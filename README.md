# Introvert Place

A digital sanctuary designed for introverts - a space to connect, create, and recharge on your own terms.

## Features

- **Connect** - Find communities and like-minded souls without the pressure
- **Creative Space** - Express yourself through art, writing, and ideas
- **Wellness** - Tools for mental health and social battery management
- **Sanctuary** - Your personal quiet space
- **Profile** - Customize your introvert identity

## Tech Stack

- HTML/CSS/JavaScript
- Supabase (Database & Auth)
- Vercel (Hosting)

## Development

```bash
# Start local server
python3 -m http.server 8080

# Deploy to Vercel
npx vercel --prod
```
